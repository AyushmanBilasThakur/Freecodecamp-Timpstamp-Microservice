Timestamp API: Microservice

This is a simple API which will convert a passed date or timestamp to UNIX and UTC timings

Just hit these endpoint:

[project-url]/api/timestamp/2019-04-25

[project-url]/api/timestamp/1568711398368

You can either pass timestamp or date in yyyy-mm-dd format. If you don't pass anything you will get current servertime in response

Responses and errors

If the date string is valid the api returns a JSON having the structure {"unix": , "utc" : } e.g. {"unix": 1479663089000 ,"utc": "Sun, 20 Nov 2016 17:31:29 GMT"}

If invalid then the response will be: {"error" : "Invalid Date" }.
